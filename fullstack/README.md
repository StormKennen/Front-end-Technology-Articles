## 🎯 学习路径

```
Node.js 入门 (1-2 个月)
├─ Node.js 基础
├─ Express 框架
├─ 路由与中间件
└─ 基础 API 开发

Node.js 进阶 (3-6 个月)
├─ TypeScript + Node.js
├─ 数据库操作
├─ 认证授权
└─ 错误处理

Node.js 高级 (6-12 个月)
├─ NestJS 企业级框架
├─ BFF 架构
├─ 性能优化
└─ 微服务架构
```

## 🛠️ 技术栈

### Web 框架

| 框架 | 特点 | 适用场景 |
|------|------|----------|
| Express | 轻量灵活 | 小型项目、快速原型 |
| Koa | 洋葱模型 | 中间件开发 |
| NestJS | 企业级 | 大型项目、团队协作 |
| Fastify | 高性能 | 高并发场景 |

### 数据库

| 数据库 | ORM | 特点 |
|--------|-----|------|
| MongoDB | Mongoose | 文档型、灵活 |
| MySQL | Sequelize | 关系型、成熟 |
| PostgreSQL | Prisma | 关系型、强大 |
| Redis | ioredis | 缓存、快速 |
