## 🎯 如何找到好文章

### 掘金
1. 看点赞数 (100+ 通常质量有保障)
2. 看评论 (有技术讨论的评论区价值高)
3. 看作者 (关注大厂工程师和开源作者)
4. 看系列 (系列文章通常更系统)

### 知乎
1. 看赞同数 (500+ 通常质量有保障)
2. 看专业度 (优先选择从业者分享)
3. 看更新时间 (技术文章优先近 2 年)
4. 看专栏 (优质专栏内容更系统)
