## 🎯 学习路径

```
第 1 阶段：HTML 基础 (1-2 周)
├─ HTML 元素与属性
├─ 语义化标签
├─ 表单与输入
└─ SEO 基础

第 2 阶段：CSS 基础 (2-4 周)
├─ 选择器与优先级
├─ 盒模型
├─ Flexbox 布局
├─ Grid 布局
└─ 响应式设计

第 3 阶段：JavaScript 核心 (4-8 周)
├─ 变量与作用域
├─ 函数与闭包
├─ 对象与原型
├─ 异步编程 (Promise/async-await)
└─ DOM 操作
```
