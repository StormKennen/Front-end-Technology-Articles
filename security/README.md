## 📖 学习路径

```
初级安全 (1-2 个月)
├─ HTTPS/TLS 基础
├─ XSS 基础与防御
├─ Cookie 安全
└─ 输入验证

中级安全 (3-6 个月)
├─ OAuth 2.0 / OIDC
├─ JWT 安全
├─ CSP 配置
└─ CSRF 防护

高级安全 (6-12 个月)
├─ OWASP Top 10 深入
├─ JS 逆向防护
├─ 反调试技术
└─ WebAssembly 保护
```
